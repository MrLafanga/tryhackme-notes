# Linux Fundamentals Part 1

## Overview
This room introduced basic Linux navigation and file management.

# Commands Learned

### pwd
Prints the current working directory.


### ls
Lists files and directories.



### cd
Changes directories.


### whoami
Displays the current user.



## Key Takeaways

- Linux is widely used in cybersecurity.
- Understanding file navigation is essential.
- Command-line skills are fundamental for penetration testing.
